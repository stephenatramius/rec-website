QuickStart Launch

Not everyone has the time or desire to be DIY. Because sometimes you need a little help programming, Recollective offers an optional QuickStart service. The QuickStart process takes your Discussion Guide and has our team program it.

  1. Review and consultation about the discussion guide
  2. Programming, Home page and site configuration
  3. Review and change, then launch

QuickStarts can be performed for single or multiple markets and languages but do not include translation services.

Recruiting

Recollective does not have a proprietary panel. Instead we partner with panel providers and qual recruiters to fulfill recruitment requests. Whether Recollective acts as an intermediary or connects you to the recruiter, we will handle any upload / data file manipulation and integration work to ensure your participants are onboarded to the platform without friction or delay.

Panel Integration

Recollective can build a custom integration to any panel provider, customer database or portal. Using our proprietary Single Sign On API and Webhooks, Recollective can not only take in prospective participants with their segmentation and custom profile fields, but Recollective can automatically signal back which participants are accepted and have actively joined a study.

Dedicated or High Security Servers

Standard Recollective sites run in a highly secure cloud infrastructure. However, if you have a requirement for the highest possible secure configuration (e.g. for healthcare or financial service research), ask about the Recollective Dedicated or High Security Services. With these optional technical setups, we provide get dedicated servers, encrypted backups and HIPAA-level protection along with GDPR compliance to satisfy the toughest IT Security needs.
